# Endorsed Forks

Forks are alternative versions of **Watchtower** with exclusive features or a specific content focus.

::: info About forks
These forks are based on **Watchtower** and are officially endorsed by the team. They share the same core codebase but target a specific use case.
:::

<VPHomeFeatures />

## Why use a fork?

If you only watch anime (and never read manga or light novels), **Animiru** provides a leaner, anime-only experience. If you want everything — anime, manga, and light novels — stick with **Watchtower**.
